# Tiva TM4C123GH6PM Mikrodenetleyici Ödevleri

**Ad Soyad:** Sude Sedef Akbulak  
**Ders:** Mikroişlemciler  
**Konu:** LCD, Timer, ADC ve UART Uygulamaları

Bu depo dönem içi mikroişlemci dersi kapsamında hazırlanan 4 ana ödevi içermektedir.

## İçerik

- **Ödev 1 – LCD Driver:**  
  16x2 LCD ekranın 4-bit modda sürülmesi.

- **Ödev 2 – LCD Saat:**  
  Timer interrupt kullanılarak LCD üzerinde saat gösterimi.

- **Ödev 3 – LCD ADC:**  
  Potansiyometreden okunan analog değerin ADC ile ölçülüp LCD’de gösterilmesi.

- **Ödev 4 – UART & GUI:**  
  Tiva C ile bilgisayar arasında UART haberleşmesi ve SharpDevelop GUI kontrolü.

> Seri haberleşme kart üzerindeki USB port üzerinden yapılmıştır, bu nedenle Ödev 4’te ek donanım bağlantısı bulunmamaktadır.
